Tic Tac Toe (CSI 3140 Assignment)

  • Implemented a 3x3 Tic Tac Toe game that can be played on the same device.
  • Designed a creative interface mimicking a paper game of Tic Tac Toe.
  • Stored information about each game, game session, and all-time best scores in PostgreSQL database.
  • Developed using HTML, CSS, and JavaScript, PHP, and PostgreSQL.

    • Prerequisites

  • Confirm PHP is installed: php --version
  • Confirm PostgreSQL is installed: psql --version

    • Initialize Database

  • run: psql -c CREATE DATABASE tictactoe
  • run: CREATE TABLE leaderboard(name varchar(50), score int);

    • Starting the Game

  • Clone this repo
  • run 'php -S localhost:8000'
  • Open the browser and go to this site http://localhost:8000/index.html
